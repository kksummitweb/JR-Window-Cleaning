# JR Window Cleaning

Premium multi-page static HTML website for JR Window Cleaning.

## Pages

- `index.html`
- `services.html`
- `about.html`
- `gallery.html`
- `free-quote.html`

## How To Run With Live Server

1. Install the **Live Server** extension in VS Code or Cursor.
2. Right-click `index.html`.
3. Click **Open with Live Server**.

The website will open in your browser.

## Tech

- HTML
- CSS
- JavaScript

No Node.js, npm, Next.js, or build step required.

## Notes

The contact and quote forms are front-end demos. Connect them to Formspree, Netlify Forms, EmailJS, or another form backend to receive real submissions.
